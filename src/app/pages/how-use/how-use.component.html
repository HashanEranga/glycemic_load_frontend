How to Use Predict GL

A tool designed to help you calculate and understand the Glycemic Load (GL) of various foods and meals.

1. Search for Common Sri Lankan Foods

Find GL values for about 250 Sri Lankan food items.

  • Navigate to "Sri Lankan Food Composition".
  • Enter a food name (e.g., "Rice, Red Raw").
  • Click "Search" to view the GL value.
  • Enter the portion size (in grams) and click "Calculate GL".
  • View the GL value and interpretation (Low, Medium, or High).
2. Predict GL for Processed Foods

Calculate GL using nutritional data from food labels.

  • Go to "Processed Foods".
  • Enter:
    • Carbohydrates (g)
    • Protein (g)
    • Fat (g)
    • Dietary Fiber (g)
  • Enter portion size and click "Calculate GL".
  • View the calculated GL value.
3. Calculate GL for Mixed Meals

Calculate GL for meals by adding multiple food items.

  • Go to "Mixed Meals".
  • Search and add food items with portion sizes.
  • Click "Add to Meal" for each item.
  • Click "Calculate GL" for total meal GL.
4. Researcher Feature (Save GI Data)

Save and manage Glycemic Index (GI) data for research.

  • Go to "Researcher" section.
  • Enter:
    • Food Name
    • Glycemic Index (GI) Value
    • Reference (e.g., research paper, study, source)
  • Click "Save to Database".
  • View, edit, or delete saved data as needed.
